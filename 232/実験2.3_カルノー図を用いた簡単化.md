# 実験2.3 カルノー図を用いた簡単化

## 問題 f = a'b'c'd' + ab'c'd' + a'b'cd' + ab'cd'

### 簡単化前の論理式
```math
f = a'b'c'd' + ab'c'd' + a'b'cd' + ab'cd'
```

### カルノー図を用いた簡単化


|ab\cd| 00 | 01 | 11 | 10 |
|:---:|:--:|:--:|:--:|:--:|
| 00  | 1  | 0  | 0  | 1  |
| 01  | 0  | 0  | 0  | 0  |
| 11  | 0  | 0  | 0  | 0  |
| 10  | 1  | 0  | 0  | 1  |


```
a' b' c' d'
a' b' c  d'
a  b' c' d'
a  b' c  d'
----------
   b'    d'


f = b' d'
```

### 簡単後の論理式
```math
f = b' d'
```
