# 実験1 3の簡単化

## 公理定理を用いた簡単化

### 簡単化前の論理式
```
S = A B' + A' B + A B
```

### 公理定理を用いた簡単化

```
S = A B' + A' B + A B
  = A ( B' + B ) + A' B
  = A + A' B
  = A + A' ( B + A' B)
  = A + A' B +
```

### 簡単化後の論理式
```
S = A + B
```
